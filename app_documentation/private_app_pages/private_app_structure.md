# Application Layout Structure

## Left Sidebar

**Logo** (links to home)

**Navigation Menu:**

- Job Matches
- Applications
- Profile
- Search Criteria

**User Profile Section** (avatar, name)

- Dropdown menu on click:
  - Settings
  - Plan
  - Logout

## Page Content

- **Page Title**
- **Page Subtitle** (optional context/description)
- **Main Content Area** (dynamic based on current page: Job Matches, Applications, Profile, etc.)
